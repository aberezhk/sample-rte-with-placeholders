Example of content-editable div container with an option to include values from input boxes

  • When editor box is empty new values will be inserted as li elements
  • When editor is not empty elements will be inserted normally
  • To insert a value from input, add some value to input box and click check-box next to it => inserted text is blue
  • Value is inserted to the last caret location (place of last click) within the editor
  • Change value in input box => value in editor is updated automatically and remains blue
  • Change inserted value in the editor => value is now pink and updates from input box will be disregarded
  • Update value so that one in input box matches the one in the editor => value is blue and linked again
  • To remove the inserted value uncheck corresponding checkbox
